# Multilingual Root-Recovery Translation of the Qur'an

1. The English translation in `en/` is the reference. It has undergone root-recovery work and its choices should guide all other languages.
2. When translating into a target language, follow the English translation's root-recovery choices — not the consensus of existing translations in that language. Target-language consensus is often the problem, not the solution.
3. Annotate translation choices in separate comment files (`comments/NNN-comments.md`).
4. Stick as close as possible to the structure of the original Arabic, never ignoring particles.
5. Use the appropriate sacred/formal register for each language (KJV English, Statenvertaling Dutch, vos/vosso Portuguese, formal engkau Indonesian, literary Esperanto, literary Frisian).
6. Whenever a conventional term appears in target-language translations (e.g. Dutch "Hiernamaals" for الآخرة, "Hel" for جهنم), check what the English root-recovery translation uses and follow that instead.
7. Arabic loanwords in the target language (especially Indonesian: kafir, muslim, surga) carry conventional meanings — use native equivalents that preserve the root meaning.
8. Look out for puns and root connections; document them in comments.
9. Prefer an integrative reading: understand surahs and verses in the context of each other as a unified whole.
